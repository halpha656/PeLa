プロペラ設計プログラム使用法

計算モード
・profile設計から -> 普通の設計
・root設計のみ    -> 根元の部分の形状設計。
・性能計算のみ    -> 設計済みのプロペラの性能計算

"root設計のみ"や"性能計算のみ"はprofile設計が終わっていないとできません。


＜profile設計＞(空力設計）

主要部の翼弦長分布と捩り角を計算で最適化します

入力パラメータ（新規設計の場合）
	・プロペラ名 propname
		これはファイル名に使われるため変な記号は使用不可
	・空気密度
	・動粘性係数
	・機体巡航速度
	・入力トルクパワ
	・回転数
	・回転半径　R
		回転の中心から翼端までの長さ
	・設計内側半径　R0
		r = R0 〜 R の部分を空力的に評価
		０にすると収束しない
	・翼型データファイル名
		使用する翼型のデータが書き込まれたテキストファイル
		プログラム用に規格化されたもののみ
		実行系(PeLa.exe)と同じディレクトリにあるフォルダ"airfoil"中にあるファイル
	・有効迎角
		半径方向にわたって一定　分布はつけられない

"新規設計"以外はこれらのパラメータはソースファイル中にすでに書き込まれています。
		
空力設計は時間がかかります。（長いと１０分超える）
しかも収束する範囲も狭いです。発散に向かい始めても何も警告はありません。
chordがガタつき始めたらもうダメです。プログラムを強制終了してください。

無事に収束した場合、propname_aero.csvというファイルが作成されます。（propnameは最初に入力したプロペラ名）
このファイルに計算結果が書き込まれています。Excel等で開くと見やすいです。


＜root設計＞

設計内側半径R0より更に内側の形状を決定し、リブ作り用のAutoCAD専用スクリプトを作成します

入力パラメータ
	・root最内半径　R00
		肉がつく最も内側の半径位置
		0 < R00 < R0　でなくてはだめ
	・r = R00での翼弦長
		この値と空力設計で求めた翼弦長分布をスプライン補間でつなぐ
	・root部分の翼型データファイル名
		\airfoil\rootの中にあるテキストファイル名を指定
	・桁形状
		パイプ桁のみを想定
		テーパは一定とみなすので半径位置と外径(r,φ)を2点指定
	・プランク厚み
	・余裕
		桁の上下とプランクの間の長さ
		０にすると根元の方でリブが桁穴によって前後で分断される
	・桁位置
		翼弦長の何パーセント位置に桁の中心がくるかを指定

終了すると2つのファイルpropname_geom.csvとpropname_cad.scrができます。
前者は最終的な形状データが書き込まれたもの、後者はAutoCAD(LT2000)で使用できるスクリプトファイルです。

＜スクリプトファイル使用法＞
AutoCADのコマンド上で”script”と入力し作成されたスクリプトファイルを開いてください。
スクリプトが実行されるとまず最初に作図補助設定のウィンドウが開くので必ずオブジェクトスナップを解除してください。
解除後、"OK"とするとあとはスクリプトに従って自動で作図がされます。
計算点すべてについてリブが描かれます。（リブごとにレイヤーが異なります）

終了すると再び作図補助設定ウィンドウが開きます。
オブジェクトスナップを戻したければ戻してください。


＜root設計のみ/性能計算のみ＞

これらのモードではすでにprofile設計が完了しているプロペラに対して、
propname_aero.csvからデータを読み込んで計算をします。

root設計ではpropname_geom.csvとpropname_cad.scrが「上書き」されます。
この時、以前の結果は残らないので注意してください。

性能計算は設計点と異なる環境でのプロペラの推力、必要パワ、効率等が計算できます。
結果はpropname_aero.csvに追加で書き込まれます。
